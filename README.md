
Block Outline Customizer
=======
Customize your block selection outline with RGB colors, opacity, and more!

Tired of the plain black selection outline in Minecraft? Block Outline Customizer lets you personalize it with custom colors, RGB mode, adjustable transparency, and variable thickness!

Features:
============
- Full Color Control
    - Custom RGB Colors: Set any color you want (0-255 for Red, Green, Blue)
    - RGB Mode: Color shifting RGB style with adjustable speed
- Outline Customization
    - Adjustable Opacity: From fully transparent to completely opaque (0.0 - 1.0)
    - Variable Width: Thicker or thinner outlines (1.0 - 10.0)
    - Smooth Animations: Buttery smooth rainbow cycling
- Easy Configuration
    - In-Game GUI: Press a keybind to open settings (no file editing!)
    - Real-Time Updates: Changes apply immediately
    - Persistent Settings: Your preferences are saved automatically

How to Use: 
==========
Install the mod (requires NeoForge or Fabric)  
Launch Minecraft and load into a world  
Press O (default keybind) to open the settings menu  
Customize your outline colors, opacity, and width  
Enable RGB mode for RGB colors!  
Click Done - your settings are saved automatically  
Change Keybind: Options → Controls → Key Binds → Block Outline Settings

Version Support:
==========
The mod is compatible with Neoforge and Fabric from version 1.21.1 to 1.21.11.

Installation
==========
Requirements:  
NeoForge or Fabric (for your Minecraft version)  
Minecraft 1.21.1 - 1.21.11  
The .jar file  
Steps:  
Download and install NeoForge or Fabric  
Download the mod for your Minecraft version  
Place the .jar file in your mods folder  
Launch Minecraft with the correct profile  
Enjoy!  



Keybinds
==========
O - Open Block Outline Settings GUI  

Keybind can be changed in Minecraft's controls menu
